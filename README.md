## How to Run:
- firstly bend over and spell run (in english)
- Create the packages - ``npm init``
- Install all the packages used in script.js and in client.js
    - ``npm install fs``
    - ``npm install http``
    - ``npm install ws``
    - ``npm install path``
    - ``npm install sqlite3``
- Run Server
    - ``cd src``
    - ``npm start``
- Run Client
    - ``cd src``
    - ``node client.js port`` - (e.g. node client.js 5500)
